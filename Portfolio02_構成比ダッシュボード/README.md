# Portfolio02｜構成比ダッシュボード（業種別）

**使用ツール：** Looker Studio  
**主題：** 構成比分析  
**作成期間：** 2025年10月  

---

## 📊 概要
業種・カテゴリ・地域別の構成比を多層的に可視化したダッシュボード。  
円グラフ／積み上げ棒グラフ／ランキング構成を用いて、  
全体バランスと個別要素を同時に俯瞰できるよう設計。

---

## 🧩 学び・改善メモ
- 凡例の統一ルール確立（カテゴリ別配色）  
- グラフ間の余白・レイアウト調整  
- 数値ラベルと割合の両立方法を実験中
